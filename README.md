# 深汕特别合作区 H5 宣传页面

一个基于 Vue 3 + TypeScript + Swiper + GSAP 的纯前端H5宣传页面项目，展示深汕特别合作区的产业发展、城市建设和乡村振兴成果。

## 技术栈

- **Vue 3** - 使用组合式 API (`<script setup>`)
- **TypeScript** - 类型安全
- **Vite** - 快速构建工具
- **Vue Router 4** - 路由管理
- **Swiper 11** - 全屏滑动交互
- **GSAP** - 强大的动画库
- **SCSS** - CSS 预处理器
- **VueUse** - Vue 组合式工具库

## 快速开始

### 安装依赖
```bash
npm install
```

### 开发模式
```bash
npm run dev
```
访问 `http://localhost:3000`

### 生产构建
```bash
npm run build
```

## 功能特性

### 1. 全屏滑动交互
- 使用 Swiper 实现竖向全屏滑动
- 支持鼠标滚轮和触摸滑动
- 右侧分页器指示当前页面

### 2. 六个核心板块
- 板块1：山海新城，产业高地
- 板块2：深汕速度，跨越发展
- 板块3：深汕智造，通达全球
- 板块4：深圳品质，宜居宜业
- 板块5：以城带乡，振兴共富
- 板块6：新起点，再出发

### 3. 丰富的动画效果
- GSAP Timeline 实现多层次动画
- 数字滚动动画
- 淡入、上滑、缩放等效果
- Intersection Observer 监听元素进入视口

## 图片资源替换

项目使用渐变色占位，需替换为真实图片。图片存放在 `public/images/` 目录下，按板块分类。

详见需求文档中的图片清单。

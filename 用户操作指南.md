## ❓ 常见问题

### Q1: 图片显示不出来怎么办？
**A**: 检查以下几点：
1. 图片是否放在 `public/images/section4/` 目录下
2. 文件名是否与代码中的路径一致（包括扩展名）
3. 在浏览器开发者工具的Network标签查看图片请求状态
4. 确认图片文件没有损坏，可以手动打开预览

### Q2: 打字机效果太慢/太快怎么调整？
**A**: 编辑 `src/composables/useTypewriter.ts`：
```typescript
const getAdaptiveSpeed = (textLength: number): number => {
  if (textLength > 100) return 20  // 改这个数字，越小越快
  if (textLength > 50) return 30   // 中等速度
  return 50                        // 慢速
}
```

或者在各个板块的Vue文件中直接指定速度：
```typescript
typeText(text, callback, { speed: 30 }) // 数字越小越快
```

### Q3: 板块3在小屏手机上显示不全怎么办？
**A**: 编辑 `src/views/Section3.vue`，找到这段CSS：
```scss
.chart-module {
  @media (max-width: 768px) {
    min-height: calc((100vh - 180px) / 3);

    // 进一步压缩：
    .module-header { margin-bottom: 8px; }
    .module-content { gap: 8px; }
    .text-content { font-size: 11px; }
    .chart-bars { height: 100px; }
  }
}
```

### Q4: 板块1魔方轮播卡顿怎么办？
**A**: 如果在低端设备上卡顿，可以改用普通slide效果：
编辑 `src/views/Section1.vue`，找到Stage 2的Swiper配置：
```vue
<swiper
  :effect="'slide'"  <!-- 改为slide -->
  <!-- 删除cube-effect配置 -->
>
```

### Q5: 想改回旧版本怎么办？
**A**: 所有旧版本文件已备份：
```bash
# 例如恢复板块1
mv src/views/Section1.vue src/views/Section1-new.vue
mv src/views/Section1-old.vue src/views/Section1.vue
```

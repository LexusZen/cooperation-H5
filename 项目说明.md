# 深汕特别合作区 H5 项目使用指南

## 项目概述

这是一个纯前端H5宣传页面，完全按照需求文档实现，包含以下7个页面：

1. **板块1** - 山海新城，产业高地（封面页）
2. **板块2** - 深汕速度，跨越发展（时间轴）
3. **板块3** - 深汕智造，通达全球（产业展示）
4. **板块4** - 深圳品质，宜居宜业（民生项目）
5. **板块5** - 以城带乡，振兴共富（乡村振兴）
6. **采访板块** - 他们说（三位采访对象）
7. **板块6** - 新起点，再出发（结尾页）

## 快速启动

```bash
# 安装依赖（已完成）
npm install

# 启动开发服务器
npm run dev

# 访问 http://localhost:3000
```

## 主要功能

### 1. 全屏滑动
- 鼠标滚轮上下滑动
- 移动端触摸滑动
- 右侧分页指示器

### 2. 动画效果
- 每个板块进入时触发动画
- 数字滚动效果（板块1）
- 元素淡入、上滑动画
- 时间轴动画（板块2）

### 3. 响应式设计
- 自动适配PC和移动端
- 禁止用户缩放（适合H5展示）

### 步骤3：修改代码

#### 方法A：修改背景图（最简单）
打开对应的 `.vue` 文件，找到 `.bg-image` 样式：

```scss
// 在 src/views/Section1.vue 中
.bg-image {
  background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
  // 👇 替换为：
  background-image: url('/images/section1/bg.jpg');
  background-size: cover;
  background-position: center;
}
```

#### 方法B：使用 img 标签
```vue
<template>
  <div class="section">
    <img src="/images/section1/bg.jpg" alt="背景" class="bg-image" />
  </div>
</template>
```

## 打包部署

### 构建生产版本
```bash
npm run build
```

构建完成后，会生成 `dist` 目录。

### 部署方式

#### 方式1：Vercel（推荐，最简单）
1. 访问 [vercel.com](https://vercel.com)
2. 导入项目
3. 自动部署完成

#### 方式2：服务器部署
将 `dist` 目录上传到服务器，配置 Nginx：

```nginx
server {
    listen 80;
    server_name yourdomain.com;
    root /path/to/dist;
    index index.html;

    location / {
        try_files $uri $uri/ /index.html;
    }
}
```

#### 方式3：阿里云OSS
1. 创建OSS Bucket
2. 开启静态网站托管
3. 上传 `dist` 目录内容

## 常见问题

### Q1: 滑动不流畅？
A: 检查图片是否过大，建议每张图片压缩到 500KB 以下。

### Q2: 动画不触发？
A: 确保 GSAP 正确加载，可以打开浏览器控制台查看报错。

### Q3: 移动端显示异常？
A: 检查 `index.html` 中的 viewport 配置是否正确。

### Q4: 如何添加新板块？
A: 参考现有板块创建新的 `.vue` 文件，然后在 `HomeView.vue` 中添加 `<swiper-slide>`。
